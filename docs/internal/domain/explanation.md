# 解説ドメインモデル

# 値オブジェクト

## 解説識別子

- 英単語解説を一意に識別する値オブジェクト
- 英単語そのものを表す

### 不変条件

- 1文字以上

## 頻出レベル (FrequencyLevel)

- 以下の値を列挙する
  - よく使う（often）
  - 使う（sometimes）
  - あまり使わない（rarely）
  - 全然使わない（hardlyEver）

## 頻出度（Frequency）

|フィールド名 |種別 |保持数 | 備考|
|-|-|-|-|
|level | 頻出レベル| 1 | |
| reason | 理由 | 1 | |

### 不変条件

- 理由は1文字以上255文字以下

## 知的レベル（SophisticationLevel）

- 以下の値を列挙する
  - かなり知的（advanced）
  - 知的（intermediate）
  - 少し知的（basic）
  - あまり知的ではない（veryBasic）

## 知的度（Sophistication）

|フィールド名 |種別 |保持数 | 備考|
|-|-|-|-|
| level | 知的レベル| 1 | |
| reason | 理由 | 1 | |

## 習熟度（Proficiency）

- 話せる（Fluent）
- 定着（Internalized）
- インプット済み（Learned）
- インプット中（Learning）

## 意味（meaning）

|フィールド名 |種別 |保持数 | 備考|
|-|-|-|-|
| values | 単語の意味のリスト | 1..n| |
| situation | 使用する状況 | 1 | |
| nuance | ニュアンス | 1 | |

### 不変条件

- 使用する状況は1文字以上255文字以下
- ニュアンスは1文字以上255文字以下

## 発音記号（PhoneticSymbols）

|フィールド名 |種別 |保持数 | 備考|
|-|-|-|-|
| weak | 弱形 | 1| |
| strong | 強形 | 1 | |

### 不変条件

- 弱形は1文字以上255文字以下
- 強形は1文字以上255文字以下

## 発音（Pronunciation）

- 米国の発音を表す

|フィールド名 |種別 |保持数 | 備考|

## コロケーション（Collocation）

- 英単語のコロケーションを表す

|フィールド名 |種別 |保持数 | 備考|
|-|-|-|-|

### 不変条件

- コロケーション内容は1文字以上255文字以下
- 意味は1文字以上255文字以下

## 例文（ExampleSentence）

## 語源（Etymology）

- 1文字以上255文字以下の文字列

# 集約

## 解説（Explanation）

- 英単語の解説を表す集約

|フィールド名 |種別 |保持数 | 備考|
|-|-|-|-|
| identifier | 解説識別子 | 1| |
| meaning | 意味 | 1 | |
| image | 視覚的イメージ | 0..1 | |
| pronunciation   | 発音 | 1 | |
| frequency |頻出度 | 1  | |
| sophistication | 知的度 | 1  | |
| collocations| コロケーションのリスト | 1..10  | |
| examples | 例文のリスト | 1..3  | |
| etymology| 語源 | 1 | |
| similarities | 類似表現のリスト | 1..5 | |
| timeline | タイムライン | 1 | |

### 振る舞い

- 画像を生成する（GenerateImage）
  - 画像生成イベントを発行する
  - すでに画像を持つ場合は例外を発する

- 画像を再生成する（RegenerateImage）
  - 画像再生成イベントを発行する
  - 画像を持たない場合は例外を発する

### リポジトリ

解説リポジトリ（ExplanationRepository）

- 識別子で集約を単一取得する（find）
- 検索条件に合致する集約の一覧を取得する（search）
- 識別子を指定して集約を破棄する（terminate）
- 集約を永続化する（persist）
